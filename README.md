
# Next Js Features

Test the features of the next js. 



## Installation

Install my-project with npm

```bash
  npm install nextjs
  cd nextjs
```
    
## Run Locally

Clone the project

```bash
  git clone https://github.com/mdshadikco/nextjs.git
```

Go to the project directory

```bash
  cd nextjs
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
